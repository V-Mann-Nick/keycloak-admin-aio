Keycloak admin aio¶

This package provides a asynchronous Keycloak Admin API wrapper.

Reference¶

class keycloak_admin_aio.KeycloakAdmin¶

Base class for Keycloak Admin API endpoints.

It handles the access_token and guarantees it being valid when using the get_access_token method or accessing a protected Keycloak resource.
